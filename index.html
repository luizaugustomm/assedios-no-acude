O projeto

Este é um projeto de doutorado do laboratório Analytics (UFCG) que tem por objetivo criar representações de dados que gerem engajamento da população a respeito de temas relevantes à comunidade. Nesta intervenção, pretendemos levantar a discussão sobre o assédio de mulheres no Açude Velho.


Chega de Fiu Fiu

O Chega de Fiu Fiu é um site, criado pelo Think Olga, com o intuido de mapear os assédios que acontecem em todo o Brasil. Para fazer uma denúncia basta escolher o lugar onde o fato aconteceu e descrever o assédio.
