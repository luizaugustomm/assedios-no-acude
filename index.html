Bem-vinda!

Essa é uma página para coletar histórias de assédio a mulheres no Açude Velho e ajudar a diminuí-las. Se você veio relatar uma dessas histórias, muito obrigada. E sentimos muito pelo que aconteceu.


Veja a seguir como fazer a denúncia do assédio.

Encontrar o açude

1. Ao entrar no site, clique em Compartilhe sua história e procure por Açude Velho.

Marcar o local do assédio

2. Arraste o pin para o local próximo ao açude onde você foi assediada. Dica: existem alguns pontos de referência no mapa para você se situar, como o Bob's, por exemplo.

Relatar o assédio

3. Informe o tipo de assédio que você pensa ter sofrido, a data e o período que ocorreu e como foi a situação. Os outros campos são opcionais. É necessário também responder a pergunta de verificação de segurança. Ao finalizar, clique em Enviar.

O projeto

Este é um projeto de doutorado do laboratório Analytics (UFCG) que cria representações de dados sobre temas socialmente relevantes. Nesta intervenção, pretendemos tornar visíveis os assédios que acontecem no dia a dia das mulheres que passam pelo Açude Velho para promover o debate e a conscientização sobre este tema. Para isso, utilizaremos os dados provenientes das denúncias do site Chega de Fiu Fiu.

Chega de Fiu Fiu é uma campanha de combate ao assédio sexual em espaços públicos lançada pelo Think Olga. Utilizaremos os dados geográficos e socio-econômicos disponibilizados pelo site para gerar uma representação física dos assédios no açude. Portanto, quando você fizer sua denúncia e especificar o local no açude onde ela ocorreu, você estará nos ajudando a coletar dados para gerar a intervenção.
